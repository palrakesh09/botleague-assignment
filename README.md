# BotLeague Frontend Assignment

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite

## Features

- Responsive Navbar
- Hero Section
- Competitions
- User Journey
- About BotLeague
- Categories
- Competition Disciplines
- League Advantages
- Join Ecosystem
- Sponsors
- Footer

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
